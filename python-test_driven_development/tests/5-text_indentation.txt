The ``5-text_indentation`` module
==================================

Using ``text_indentation``
--------------------------

This is a doctest file for the text_indentation function.

First, import the function:

    >>> text_indentation = __import__('5-text_indentation').text_indentation

Test with simple text containing periods:

    >>> text_indentation("Hello. World.")
    Hello.
    <BLANKLINE>
    World.
    <BLANKLINE>

Test with text containing question marks:

    >>> text_indentation("How are you? Still alive?")
    How are you?
    <BLANKLINE>
    Still alive?
    <BLANKLINE>

Test with text containing colons:

    >>> text_indentation("Name: Uliana. Age: 26.")
    Name:
    <BLANKLINE>
    Uliana.
    <BLANKLINE>
    Age:
    <BLANKLINE>
    26.
    <BLANKLINE>

Test with all three punctuation marks:

    >>> text_indentation("Hello: How are you? I am fine.")
    Hello:
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    I am fine.
    <BLANKLINE>

Test with spaces after punctuation (should be removed):

    >>> text_indentation("Hello.   World.")

    <BLANKLINE>

    Answer.
    <BLANKLINE>

    <BLANKLINE>
    Content.
    <BLANKLINE>

Test with text without special punctuation:

    >>> text_indentation("Hello World")

Test with empty string:

    >>> text_indentation("")

Test with only punctuation:

    >>> text_indentation(".")

    <BLANKLINE>

    >>> text_indentation("?")

    <BLANKLINE>

    >>> text_indentation(":")

Test with consecutive punctuation:

    >>> text_indentation("End.?:")

    <BLANKLINE>
    ?

    :
    <BLANKLINE>

Test with punctuation at the beginning:

    >>> text_indentation(".Hello")

    Hello

    >>> text_indentation("?Hello")
    ?

    Hello

    >>> text_indentation(":Hello")

Test with multiple spaces after punctuation:

    >>> text_indentation("Hello.     How are you?     Good.")

Test with tabs and spaces after punctuation:
